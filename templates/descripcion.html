Segmentación de Productos Amazon con K-Means

Agrupamiento no supervisado sobre el catálogo de Amazon India.

Planteamiento del problema

Plataformas de comercio electrónico como Amazon generan una gran cantidad de transacciones con una variedad significativa de productos que oscilan entre diferentes precios, categorías, valoraciones de clientes y popularidad. Esta información se acumula en bases de datos masivas que, sin el procesamiento adecuado, resultan poco útiles para la toma de decisiones estratégicas.

El problema radica en que los datos recolectados no son aprovechados mediante métodos de minería de datos que permitan encontrar información oculta de utilidad para estrategias de precios, campañas de descuentos y recomendaciones personalizadas. Por ende: ¿es posible identificar grupos naturales o segmentos de mercado que compartan características similares de precio, descuento y popularidad?

Este proyecto aplica K-Means para segmentar automáticamente los productos del dataset de Amazon en grupos homogéneos, permitiendo identificar patrones comerciales y apoyar decisiones de marketing.

Objetivos

  • Preparar y limpiar el dataset de Amazon Sales para análisis numérico.
  • Seleccionar las variables más representativas del perfil comercial de cada producto.
  • Justificar el número de clústeres con el método del codo.
  • Visualizar e interpretar los segmentos resultantes.
  • Extraer conclusiones de negocio accionables a partir del agrupamiento.

Metodología

Carga y limpieza. Eliminación de símbolos monetarios, caracteres especiales y valores nulos del CSV original.

Selección de variables. Elección de las 5 variables numéricas que mejor capturan el perfil comercial.

Normalización. Estandarización con StandardScaler para que todas las variables tengan igual peso en la distancia euclidiana.

Método del codo. Ejecución de K-Means para K de 1 a 10, evaluando la inercia para seleccionar el K óptimo.

K-Means con K=4. Entrenamiento del modelo final y asignación de etiquetas a cada producto.

Reducción PCA. Proyección a 2D para visualización de los clústeres en gráfica de dispersión.

Interpretación. Análisis del perfil de cada clúster a partir de los centroides y estadísticas descriptivas.
